# Customer Churn Prediction (Deep Learning + TensorFlow + Streamlit)

This project predicts **customer churn** (whether a customer is likely to leave a service) using a **Deep Learning model built with TensorFlow/Keras**.  
A simple and interactive **Streamlit web app** is included for real-time predictions.
